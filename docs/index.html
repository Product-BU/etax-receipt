ETDA Compliant for Thailand

Modern E-Tax & E-Receipts, Made Simple.

Open E-Tax Receipt is a free, open-source, and self-hostable solution to generate and manage electronic tax invoices and receipts, fully compliant with Thai regulations.

Get Started Learn More

Powerful Features, Zero Cost

Everything you need to streamline your invoicing process.

ETDA Compliant

Generates PDF and XML files for all documents, adhering to Thailand's regulatory standards.

REST API

Integrate with your existing systems like ERP or accounting software for seamless workflows.

Modern Admin Portal

A clean and responsive dashboard built with Django and AdminLTE for easy management.

Bulk CSV Import

Save time by creating hundreds of documents at once by simply uploading a CSV file.

Analytics Dashboard

Visualize your data with document statistics and charts to gain valuable insights.

Multiple Document Types

Supports Tax Invoices, Receipts, Credit Notes, and Debit Notes out of the box.
